# CoreWebView2OriginFeatureState Enum

Specifies the state of an origin feature.

| Name |  Value | Description |
|--|--|--|
|`Enabled` | 0x0  |  Sets the enabled state of the origin feature.|
|`Disabled` | 0x1  |  Sets the disabled state of the origin feature.|
