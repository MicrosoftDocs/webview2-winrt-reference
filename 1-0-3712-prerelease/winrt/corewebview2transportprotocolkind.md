# CoreWebView2TransportProtocolKind Enum

Specifies the transport protocol for port configuration.

| Name |  Value | Description |
|--|--|--|
|`Udp` | 0x0  |  User Datagram Protocol - fast, connectionless protocol.
UDP is a connectionless protocol that is commonly used for applications that require fast, efficient transmission of data and can tolerate some data loss. WebRTC commonly uses UDP for media streaming.
|
